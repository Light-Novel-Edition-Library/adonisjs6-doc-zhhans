# 常见问题 (FAQs)

## 谁维护 AdonisJS？

AdonisJS 是一个独立项目，由 Harminder Virk 于 2015 年创建。该框架由核心团队和社区贡献者积极维护。

框架创建者 (Harminder Virk) 是项目负责人，并全职致力于该框架的工作。

该项目通过 GitHub Sponsors 获得资助。如果您或您的企业从 AdonisJS 中受益，请考虑赞助我们以支持框架的发展。

## AdonisJS 如何授权？

AdonisJS（框架）和官方软件包是在 MIT 许可证下分发的。此外，源代码在 GitHub 上公开可用。

## AdonisJS 可靠且维护良好吗？

AdonisJS 已被 Marie Claire、Cleavr、Ledger、Cavai、Kayako、Renault Group、Zakodium、FIVB 等许多公司在不同程度上用于生产环境。

框架创建者全职致力于 AdonisJS，确保框架得到积极改进和维护。

在 v6 版本发布期间，我们迁移到了 ES 模块系统。经 Snyk 安全扫描报告，官方维护的软件包零安全漏洞。我们持续致力于编写更好的文档、发布新功能并改进现有代码库。

## AdonisJS 快吗？

在创建框架或添加新功能时，我们主要专注于解决现实世界的问题，而不是为了让 AdonisJS 赢得基准测试竞赛而削减功能。

然而，我们确实密切关注性能指标，并在重要的地方微调框架性能。例如：

- 独立模式下的 AdonisJS HTTP 服务器在性能上与 Fastify 相当。
- 框架的验证层优于 Node.js 生态系统中其他流行的验证库。
