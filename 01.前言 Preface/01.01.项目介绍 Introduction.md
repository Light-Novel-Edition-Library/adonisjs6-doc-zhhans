# 介绍 (前言)

## 什么是 AdonisJS？

AdonisJS 是一个基于 Node.js 的**TypeScript 优先**的 Web 框架。你可以用它来创建全栈 Web 应用或 JSON API 服务器。

从根本上说，AdonisJS 为你的应用提供了结构，配置了无缝的 TypeScript 开发环境，为你的后端代码配置了 HMR（热模块替换），并提供了大量维护良好且文档详尽的软件包。

我们设想使用 AdonisJS 的团队可以花费更少的时间在琐碎的决策上（比如为每个小功能挑选 npm 包、编写胶水代码、争论完美的文件夹结构），从而将更多的时间用于交付对业务需求至关重要的实际功能。

### 前端无关 (Frontend agnostic)

AdonisJS 专注于后端，让你自由选择你喜欢的前端技术栈。

如果你喜欢保持简单，可以将 AdonisJS 与传统的模板引擎搭配使用，在服务器端生成静态 HTML；或者为你的 Vue/React 前端应用创建 JSON API；亦或是使用 Inertia 让这个框架与你最爱的前端框架完美协作。

AdonisJS 旨在为你提供“自带电池”（batteries-included）的体验，从零开始构建稳健的后端应用。无论是发送电子邮件、验证用户输入、执行 CRUD 操作，还是验证用户身份，我们都为你处理好了。

### 现代且类型安全 (Modern and Type-safe)

AdonisJS 建立在现代 JavaScript 原语之上。我们使用 ES 模块、Node.js 子路径导入别名、用于执行 TypeScript 源码的 SWC，以及用于资产打包的 Vite。

此外，在设计框架 API 时，TypeScript 发挥了重要作用。例如，AdonisJS 拥有：

  * 类型安全的事件触发器 (Event emitter)
  * 类型安全的环境变量
  * 类型安全的验证库

### 拥抱 MVC (Embracing MVC)

AdonisJS 拥抱经典的 MVC 设计模式。你首先使用函数式 JavaScript API 定义路由，将控制器绑定到路由上，并在控制器内编写处理 HTTP 请求的逻辑。

**start/routes.ts**

```typescript
import router from '@adonisjs/core/services/router'
const PostsController = () => import('#controllers/posts_controller')

router.get('posts', [PostsController, 'index'])
```

控制器可以使用模型（Model）从数据库获取数据，并渲染视图（即模板）作为响应。

**app/controllers/posts\_controller.ts**

```typescript
import Post from '#models/post'
import type { HttpContext } from '@adonisjs/core/http'

export default class PostsController {
  async index({ view }: HttpContext) {
    const posts = await Post.all()
    return view.render('pages/posts/list', { posts })
  }
}
```

如果你正在构建 API 服务器，你可以将视图层替换为 JSON 响应。但是，处理和响应 HTTP 请求的流程保持不变。

**app/controllers/posts\_controller.ts**

```typescript
import Post from '#models/post'
import type { HttpContext } from '@adonisjs/core/http'

export default class PostsController {
  async index({ view }: HttpContext) {
    const posts = await Post.all()
    // return view.render('pages/posts/list', { posts })

    /**
     * posts 数组将自动序列化为 JSON。
     */
    return posts
  }
}
```

## 指南假设 (Guides assumptions)

AdonisJS 文档是作为参考指南编写的，涵盖了核心团队维护的多个软件包和模块的使用方法及 API。

本指南**不会**教你如何从零开始构建一个应用程序。如果你在寻找教程，我们建议从 **Adocasts** 开始你的旅程。Tom（Adocasts 的创建者）制作了一些高质量的播客，帮助你迈出使用 AdonisJS 的第一步。

话虽如此，文档广泛涵盖了可用模块的使用以及框架的内部工作原理。
