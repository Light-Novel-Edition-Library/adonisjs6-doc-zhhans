# Cookie (Cookies)

在 HTTP 请求期间，**请求 Cookie** 会自动被解析。您可以使用 [**`request`**](https://www.google.com/search?q=./request.md) 对象读取 Cookie，并使用 [**`response`**](https://www.google.com/search?q=./response.md) 对象设置/清除 Cookie。

```ts
// title: 读取 Cookie
import router from '@adonisjs/core/services/router'

router.get('cart', async ({ request }) => {
  // 使用 request.cookie(key, defaultValue) 读取 Cookie，如果不存在则返回默认值
  const cartItems = request.cookie('cart_items', []) 
  console.log(cartItems)
})
```

```ts
// title: 设置 Cookie
import router from '@adonisjs/core/services/router'

router.post('cart', async ({ request, response }) => {
  const id = request.input('product_id')
  // 使用 response.cookie(key, value) 设置签名 Cookie
  response.cookie('cart_items', [{ id }]) 
})
```

```ts
// title: 清除 Cookie
import router from '@adonisjs/core/services/router'

router.delete('cart', async ({ request, response }) => {
  // 使用 response.clearCookie(key) 清除 Cookie
  response.clearCookie('cart_items') 
})
```

## 配置 (Configuration)

用于设置 Cookie 的默认配置定义在 **`config/app.ts` 文件**中。您可以根据应用程序的要求自由调整这些选项。

```ts
http: {
  cookie: {
    domain: '',
    path: '/',
    maxAge: '2h', // Cookie 的最大有效期
    httpOnly: true, // 仅限服务器访问
    secure: true, // 仅限 HTTPS
    sameSite: 'lax', // SameSite 策略
    /**
     * 实验性属性
     * https://developer.mozilla.org/en-US/docs/Web/HTTP/Headers/Set-Cookie#partitioned
     */
    partitioned: false,
    priority: 'medium',
  }
}
```

这些选项将转换为 [**`Set-Cookie` 属性**](https://www.google.com/search?q=%5Bhttps://developer.mozilla.org/en-US/docs/Web/HTTP/Headers/Set-Cookie%23attributes%5D\(https://developer.mozilla.org/en-US/docs/Web/HTTP/Headers/Set-Cookie%23attributes\))。`maxAge` 属性接受一个**基于字符串的时间表达式**（例如 `'2h'`），其值将被转换为**秒**。

设置 Cookie 时，可以**覆盖**相同的选项集。

```ts
response.cookie('key', value, {
  domain: '',
  path: '/',
  maxAge: '2h', // 覆盖默认值
  httpOnly: true,
  secure: true,
  sameSite: 'lax',
})
```

## 支持的数据类型 (Supported data types)

Cookie 值使用 **`JSON.stringify` 序列化为字符串**；因此，您可以使用以下 JavaScript 数据类型作为 Cookie 值。

  * string
  * number
  * bigInt
  * boolean
  * null
  * object
  * array

<!-- end list -->

```ts
// Object
response.cookie('user', {
  id: 1,
  fullName: 'virk',
})

// Array
response.cookie('product_ids', [1, 2, 3, 4])

// Boolean
response.cookie('is_logged_in', true)

// Number
response.cookie('visits', 10)

// BigInt
response.cookie('visits', BigInt(10))

// Date 对象被转换为 ISO 字符串
response.cookie('visits', new Date())
```

## 签名 Cookie (Signed cookies)

使用 **`response.cookie` 方法**设置的 Cookie 是**签名**的。签名 Cookie 是**防篡改**的，这意味着更改其内容将使签名失效，并且该 Cookie 将被忽略。

Cookie 使用 `config/app.ts` 文件中定义的 **`appKey`** 进行签名。

:::note

签名 Cookie 仍然可以通过 Base64 解码来**读取**。如果您希望 Cookie 值**不可读**，则可以使用**加密 Cookie**。

:::

```ts
import router from '@adonisjs/core/services/router'

router.get('/', async ({ request, response }) => {
  // 设置签名 Cookie
  response.cookie('user_id', 1) 

  // 读取签名 Cookie (自动验证签名)
  request.cookie('user_id') 
})
```

## 加密 Cookie (Encrypted cookies)

与签名 Cookie 不同，**加密 Cookie** 的值**无法解码为纯文本**。因此，您可以将加密 Cookie 用于包含**敏感信息**的值，这些信息不应被任何人读取。

加密 Cookie 使用 **`response.encryptedCookie` 方法**设置，并使用 **`request.encryptedCookie` 方法**读取。在底层，这些方法使用了 [加密模块](https://www.google.com/search?q=../security/encryption.md)。

```ts
import router from '@adonisjs/core/services/router'

router.get('/', async ({ request, response }) => {
  // 设置加密 Cookie
  response.encryptedCookie('user_id', 1) 

  // 读取加密 Cookie (自动解密)
  request.encryptedCookie('user_id') 
})
```

## 普通 Cookie (Plain cookies)

**普通 Cookie** 旨在用于您希望前端代码可以使用 **`document.cookie`** 值访问 Cookie 的情况。

默认情况下，我们对原始值调用 **`JSON.stringify`** 并将其转换为 Base64 编码的字符串。这样做是为了让您可以对 Cookie 值使用 `objects` 和 `arrays`。但是，**可以关闭**此编码。

普通 Cookie 使用 **`response.plainCookie` 方法**定义，并可以使用 **`request.plainCookie` 方法**读取。

```ts
import router from '@adonisjs/core/services/router'

router.get('/', async ({ request, response }) => {
  // 设置普通 Cookie
  response.plainCookie('user', { id: 1 }, {
    httpOnly: true // 仍然可以设置 httpOnly
  })

  // 读取普通 Cookie (自动 Base64 解码和 JSON 解析)
  request.plainCookie('user') 
})
```

要关闭编码，请在选项对象中设置 **`encode: false`**。

```ts
response.plainCookie('token', tokenValue, {
  httpOnly: true,
  encode: false, // 关闭编码
})

// 读取未编码的普通 Cookie
request.plainCookie('token', {
  encoded: false // 必须设置为 false 才能正确读取
})
```
